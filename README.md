# Boilerplate PHP

**OBS:** Boilerplate para desenvolvimento de aplicações php usando live-reload e sass

# Tecnologias utilizadas

- [x] PHP
- [x] HTML
- [x] JAVASCRIPT
- [x] SASS
- [x] BABEL
- [x] WEBPACK
- [x] BROWSER-SYNC
- [x] NORMALIZE.CSS
- [x] BABEL-PLUGIN-WEBPACK-ALIAS

** Run the code:**

**1.**
`Install node version 16.0.0`

**2.**
`cd build`

**3.**
`pnpm install && pnpm build`
