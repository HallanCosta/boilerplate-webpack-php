# Boilerplate PHP

**OBS:** Boilerplate para desenvolvimento de aplicações php usando live-reload e sass

# Tecnologias utilizadas

- [x] PHP
- [x] HTML
- [x] JAVASCRIPT
- [x] SASS
- [x] WEBPACK
- [x] BROWSER-SYNC

** Run the code:**

**1.**
`Install node version 16.0.0`

**2. if necessary**
`npm install -g npm@9.2.0`

**3.**
`npm install --legacy-peer-deps && npm run build`
